3 Jenis Variable di JavaScript

1. Variable Var

Var adalah variable yang dapat di ubah dan bisa di panggil baik di dalam scoops maupun diluar scoops

2. Variable Let

Let adalah variable yang dapat di ubah tetapi tidak bisa di panggil di luar scoops

3. Variable Const

Const adalah variable yang tidak dapat di ubah dan tidak dapat di panggil baik di dalam scoops maupun diluar scoops
